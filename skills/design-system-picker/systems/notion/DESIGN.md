# Notion 啟發的設計系統

> 類別：Productivity & SaaS
> 一站式工作空間。溫暖極簡、襯線標題、柔和表面。

## 1. 視覺主題與氛圍

Notion 的網站體現了工具本身的哲學：一張不阻擋你的空白畫布。設計系統建立在暖色中性而非冷灰之上，創造出獨特的平易近人的極簡——更像高品質紙張，而非無菌的玻璃。頁面畫布是純白（`#ffffff`），但文字並非純黑——而是溫暖的近黑（`rgba(0,0,0,0.95)`），不著痕跡地柔化閱讀體驗。暖灰色階（`#f6f5f4`、`#31302e`、`#615d59`、`#a39e98`）帶著微妙的黃棕底蘊，賦予介面一種有觸感、近乎類比的溫度。

自訂的 NotionInter 字型（Inter 的改良版）是系統的骨幹。在展示尺寸（64px）下，它使用激進的負字距（-2.125px），讓標題顯得壓縮而精確。字重範圍比典型系統更寬：400 用於內文、500 用於 UI 元素、600 用於半粗標籤、700 用於展示標題。較大字級啟用 OpenType 功能 `"lnum"`（lining numerals）與 `"locl"`（localized forms），增添字型上的細膩感，獎勵仔細閱讀。

讓 Notion 視覺語言獨樹一格的是它的邊框哲學。Notion 不用厚邊框或陰影，而是使用超細的 `1px solid rgba(0,0,0,0.1)` 邊框——邊框如耳語般存在，幾乎難以察覺的分隔線，在無重量下建立結構。陰影系統同樣節制：多層堆疊累積不透明度從不超過 0.05，創造出感覺得到卻看不見的深度。

**核心特徵：**
- NotionInter（改良版 Inter）展示尺寸帶負字距（64px 用 -2.125px）
- 暖色中性盤面：灰帶黃棕底蘊（`#f6f5f4` 暖白、`#31302e` 暖深）
- 近黑文字透過 `rgba(0,0,0,0.95)`——非純黑，創造微小溫度
- 超細邊框：全站使用 `1px solid rgba(0,0,0,0.1)`——耳語般的分隔
- 多層陰影堆疊、單層不透明度低於 0.05，營造若有似無的深度
- Notion Blue（`#0075de`）作為核心 UI 唯一的點綴色，用於 CTA 與互動元素
- 藥丸徽章（9999px 圓角）搭配藍色帶調背景作為狀態指示器
- 8px 基本間距單位搭配有機、非僵硬的尺度

### 原則
- **大尺寸時壓縮**：NotionInter 展示尺寸 64px 用 -2.125px 字距，逐漸放鬆到 26px 的 -0.625px，再到 16px 的 normal。壓縮在標題創造密度，內文則維持可讀性。
- **四階字重系統**：400（內文／閱讀）、500（UI／互動）、600（強調／導覽）、700（標題／展示）。比多數系統更寬的字重範圍允許細膩的階層。
- **暖度縮放**：行高隨字級增大而收緊——內文（16px）1.50、副標 1.23–1.27、展示 1.00–1.04。創造更密實、更有衝擊力的標題。
- **徽章微字距**：12px 徽章文字使用正字距（0.125px）——系統中唯一的正字距，讓小字更寬、更易讀。

## 5. 間距與佈局

### 間距系統
- 基本單位：8px
- 尺度：2px、3px、4px、5px、6px、7px、8px、11px、12px、14px、16px、24px、32px
- 非僵硬的有機尺度，含小數值（5.6px、6.4px）做微調

### 網格與容器
- 最大內容寬度：約 1200px
- Hero：置中單欄搭配寬裕的頂部 padding（80-120px）
- 功能區段：2-3 欄卡片格線
- 全寬暖白（`#f6f5f4`）區段背景做交替
- 程式碼／儀表板截圖以耳語邊框收束

### 留白哲學
- **寬裕的垂直節奏**：主要區段之間 64-120px。Notion 用大量垂直 padding 讓內容呼吸。
- **暖色交替**：白色區段與暖白（`#f6f5f4`）區段交替，創造溫和的視覺節奏，無刺眼的色彩斷裂。
- **內容優先的密度**：內文區塊密實（行高 1.50）但被寬裕邊距環繞，在留白海洋中創造可讀的內容孤島。

### 圓角尺度
- 微（4px）：按鈕、輸入、功能性互動元素
- 細緻（5px）：連結、列表項、選單項
- 標準（8px）：小卡片、容器、行內元素
- 舒適（12px）：標準卡片、功能容器、圖片頂部
- 大（16px）：Hero 卡片、精選內容、推廣區塊
- 完整藥丸（9999px）：徽章、藥丸、狀態指示器
- 圓形（100%）：頁籤指示器、頭像

## 6. 深度與層次

| 層級 | 處理方式 | 用途 |
|------|----------|------|
| 平面（Level 0） | 無陰影、無邊框 | 頁面背景、文字區塊 |
| 耳語（Level 1） | `1px solid rgba(0,0,0,0.1)` | 標準邊框、卡片外框、分隔 |
| 柔卡（Level 2） | 4 層陰影堆疊（最大不透明度 0.04） | 內容卡片、功能區塊 |
| 深卡（Level 3） | 5 層陰影堆疊（最大不透明度 0.05、52px 模糊） | 彈窗、精選面板、hero 元素 |
| Focus（無障礙） | `2px solid var(--focus-color)` 外框 | 所有互動元素的鍵盤 focus |

**陰影哲學**：Notion 的陰影系統使用多層、極低單層不透明度（0.01 到 0.05），累積成柔和、自然的層次。4 層卡片陰影從 1.04px 跨到 18px 模糊，創造的是漸層深度而非單一硬陰影。5 層深陰影在 0.05 不透明度下擴展到 52px 模糊，產生感覺像自然光而非電腦合成的環境遮蔽。這種分層做法讓元素感覺嵌入頁面之中而非漂浮其上。

### 裝飾性深度
- Hero 區段：裝飾性角色插畫（俏皮、手繪風格）
- 區段交替：白色到暖白（`#f6f5f4`）的背景轉換
- 無硬性區段邊框——分隔來自背景色變化與間距

## 7. 響應式行為

### 斷點
| 名稱 | 寬度 | 主要變化 |
|------|------|----------|
| Mobile Small | <400px | 緊湊單欄、最少 padding |
| Mobile | 400-600px | 標準手機、堆疊佈局 |
| Tablet Small | 600-768px | 開始出現 2 欄 |
| Tablet | 768-1080px | 完整卡片格線、展開 padding |
| Desktop Small | 1080-1200px | 標準桌面佈局 |
| Desktop | 1200-1440px | 完整佈局、最大內容寬度 |
| Large Desktop | >1440px | 置中、寬裕邊距 |

### 觸控目標
- 按鈕用舒適的 padding（8px-16px 垂直）
- 導覽連結 15px 搭配足夠間距
- 藥丸徽章有 8px 水平 padding 作為點擊目標
- 手機選單切換使用標準漢堡按鈕

### 摺疊策略
- Hero：64px 展示 → 縮至 40px → 手機 26px、維持等比例字距
- 導覽：水平連結 + 藍色 CTA → 漢堡選單
- 功能卡：3 欄 → 2 欄 → 單欄堆疊
- 產品截圖：以響應式圖片維持長寬比
- 信任列 logo：格線 → 手機水平捲動
- 頁腳：多欄 → 堆疊單欄
- 區段間距：80px+ → 手機 48px

### 圖像行為
- 工作空間截圖在所有尺寸維持耳語邊框
- Hero 插畫等比例縮放
- 產品截圖使用響應式圖片搭配一致的圓角
- 全寬暖白區段維持邊到邊的處理

## 8. 無障礙與狀態

### Focus 系統
- 所有互動元素接受可見的 focus 指示器
- Focus 外框：`2px solid` 搭配 focus 色 + shadow level 200
- 所有互動元件支援 Tab 導覽
- 高對比文字：白底近黑超過 WCAG AAA（>14:1 比例）

### 互動狀態
- **預設**：標準外觀帶耳語邊框
- **Hover**：文字色變化、按鈕 scale(1.05)、連結加底線
- **啟用／按壓**：scale(0.9) 變形、深色背景變體
- **Focus**：藍色外框 ring 帶陰影強化
- **Disabled**：暖灰（`#a39e98`）文字、降低不透明度

### 色彩對比
- 主要文字（rgba(0,0,0,0.95)）對白：約 18:1 比例
- 次要文字（`#615d59`）對白：約 5.5:1 比例（WCAG AA）
- 藍色 CTA（`#0075de`）對白：約 4.6:1 比例（WCAG AA 大字級）
- 徽章文字（`#097fe8`）對徽章底（`#f2f9ff`）：約 4.5:1 比例（WCAG AA 大字級）

### 迭代指南
1. 永遠使用暖色中性——Notion 的灰帶黃棕底蘊（`#f6f5f4`、`#31302e`、`#615d59`、`#a39e98`），絕不藍灰
2. 字距隨字級縮放：64px 用 -2.125px、54px 用 -1.875px、26px 用 -0.625px、16px normal
3. 四階字重：400（讀）、500（互動）、600（強調）、700（宣告）
4. 邊框是耳語：`1px solid rgba(0,0,0,0.1)`——絕不更重
5. 陰影用 4-5 層，單層不透明度永不超過 0.05
6. 暖白（`#f6f5f4`）區段背景對視覺節奏是必要的
7. 藥丸徽章（9999px）給狀態／標籤、4px 圓角給按鈕與輸入
8. Notion Blue（`#0075de`）是核心 UI 唯一的飽和色——節制用於 CTA 與連結
